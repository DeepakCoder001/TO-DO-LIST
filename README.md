# 📝 to-dolist app

A simple and responsive **To-Do List** application built using **HTML, CSS, and JavaScript**. It allows users to add new tasks and delete completed or unwanted tasks through a clean and user-friendly interface.

---

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- 📱 Responsive design
- 🎨 Simple and clean user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

```
to-dolist app/

```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/DeepakCoder001/TO-DO-LIST.git
```

### Navigate to the Project Folder

```bash
cd todolist app
```

### Run the Project

Open the `index.html` file in your web browser.
